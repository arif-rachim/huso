## âœï¸ Creating New Lessons

### JSON Lesson Format
All lessons follow the TypeScript `Lesson` interface in `src/types/lesson.types.ts`.

Example structure:
```json
{
  "day": 1,
  "subject": "biology",
  "title": "Cell Theory",
  "emoji": "ğŸ”¬",
  "learningGoal": "Understand cell theory principles",
  "estimatedTime": 60,
  "objectives": ["...", "...", "..."],
  "discoveryStory": { ... },
  "concepts": [ ... ],
  "workedExamples": [ ... ],
  "practiceProblems": [ ... ],
  "quiz": [ ... ],
  "selfAssessment": [ ... ]
}
```

### Type Safety
TypeScript ensures:
- All required fields are present
- Subjects are valid (`biology`, `chemistry`, `earth-science`, `math`)
- Quiz questions have exactly 5 options
- All data types are correct

### Adding a New Day
1. Create `data/lessons/day-XX.json`
2. Follow the `Lesson` type structure
3. Run `npm run build` to validate
4. If TypeScript compiles without errors, your lesson is valid!
