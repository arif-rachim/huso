# HUSO Preparation Platform

A comprehensive **40-day study program** designed for **Raoul Ardy** (Year 9, DIA International School) to prepare for the **Harvard Undergraduate Science Olympiad (HUSO) 2025-26 Middle East Competition**.

## 📊 Competition Overview

- **Target Student**: Raoul Ardy, Grade 9
- **Category**: Category 1 (Grades 7-8/Years 8-9)
- **Timeline**: 40 days intensive preparation
- **Subjects**: Biology, Chemistry, Earth Science, Mathematics (4 subjects)
- **Exam Format**: 3-hour online exam, multiple choice, penalty system (-0.25 for wrong answers)
- **Goal**: Qualify for top 100 to advance to Round 2 in Dubai
- **Competition Date**: November 15-16, 2025

## 🚀 Tech Stack

### TypeScript + JSON Architecture
- **TypeScript**: Type-safe lesson structure and rendering
- **JSON**: Clean, validated content storage
- **Vanilla JavaScript**: Compiled output, no framework bloat
- **CSS**: Custom styling with subject-specific themes
- **LocalStorage**: Client-side progress tracking

### Why This Approach?
- ✅ **Type Safety**: TypeScript validates all JSON lesson content at compile time
- ✅ **No Build Bloat**: No React, no webpack, just clean compiled JS
- ✅ **Easy Content Creation**: Write lessons in JSON with full IDE autocomplete
- ✅ **Offline First**: Static files, no server needed
- ✅ **Fast**: Minimal dependencies, instant page loads

## ✍️ Creating New Lessons

### JSON Lesson Format
All lessons follow the TypeScript `Lesson` interface in `src/types/lesson.types.ts`.

Example structure:
```json
{
  "day": 1,
  "subject": "biology",
  "title": "Cell Theory",
  "emoji": "🔬",
  "learningGoal": "Understand cell theory principles",
  "estimatedTime": 60,
  "objectives": ["...", "...", "..."],
  "discoveryStory": { ... },
  "concepts": [ ... ],
  "workedExamples": [ ... ],
  "practiceProblems": [ ... ],
  "quiz": [ ... ],
  "selfAssessment": [ ... ]
}
```

### Type Safety
TypeScript ensures:
- All required fields are present
- Subjects are valid (`biology`, `chemistry`, `earth-science`, `math`)
- Quiz questions have exactly 5 options
- All data types are correct

### Adding a New Day
1. Create `data/lessons/day-XX.json`
2. Follow the `Lesson` type structure
3. Run `npm run build` to validate
4. If TypeScript compiles without errors, your lesson is valid!

## 🎨 Learning Methodology

### "Explain Like I'm 5" Approach
- **45-60 minutes** daily commitment
- **Storytelling**: Every concept starts with a narrative
- **Real-world analogies**: Cities, LEGO, factories, etc.
- **Mind-blowing facts**: Specific numbers and surprising connections
- **Personal relevance**: "Why YOU should care"
- **Progressive difficulty**: Simple → Complex

### Content Types
1. **Discovery Story**: Historical context with adventure narrative
2. **Concept Boxes**: Core ideas with analogies and facts
3. **Worked Examples**: Detective-style problem solving
4. **Practice Problems**: Real-world scenarios
5. **Quick Quiz**: 5 HUSO-format questions
6. **Self-Assessment**: Reflection checklist
